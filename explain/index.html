web

  • まず、webについて説明します。
  • Web は、 World Wide Web (WWW) とも呼ばれ、インターネット上で提供される文書、画像、音声、動画などの リソース を相互に結びつけたシステムのことを指します。
  • Web は、リソースを識別するために URI (Uniform Resource Identifier) を用いています。
  • また、リソースを取得・表示するために HTTP (Hypertext Transfer Protocol) というプロトコルを用いています。

HTML

  • HTMLはウェブページのコンテンツを記述するためのマークアップ言語です。
  • HTMLはタグを用いて意味づけを行います。
  • classとidは文書中特定の箇所を表すための識別子です。
  • class:同一の文書中に同一の識別子が複数回出現可です。
  • id:同一の文書中に同一の識別子が複数回出現不可です。

CSS

  • CSSはHTMLの見た目や体裁を整える。
  • HTMLファイルからCSSファイルを呼び出します。
  • プロパティは要素にスタイルを適用する際、どのように表示するかを記述したものです。
  • CSSセレクタは、内側のセレクタが、外側のセレクタより強いです。
  • また、該当スタイルが複数ある場合、idセレクタ>classセレクタ>要素セレクタの順に強いです。

JavaScript

  • JavaScriptはウェブページのふるまいを決める言語です。
  • HTMLからJavaScriptを読み込みます。
  • JavaScriptはブラウザ上で動作する唯一の言語です。
  • JavaScriptは実行時に変数の型が決める動的型付け言語です。
